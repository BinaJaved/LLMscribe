# LLM Scribe: Code to Doc

LLM Scribe is a developer productivity tool that converts code snippets into readable documentation. It uses LLM model **PolyCoder** for generating structured technical summaries and **OpenAI** for generating plain English explanations. The project is built with a **React frontend**, a **Node.js backend**, and a **Python service** running PolyCoder locally.

## Features

- Paste any code snippet and generate documentation automatically.
- Technical summaries powered by PolyCoder.
- Human-readable explanations generated by OpenAI.
- Cleaned and filtered output that removes license headers or unwanted lines.
- Styled frontend interface with bordered sections for technical and explanation results.

## Tech Stack

- **Frontend**: React, TypeScript, ReactMarkdown, Axios
- **Backend**: Node.js, Express, TypeScript
- **Python Service**: FastAPI, Transformers (PolyCoder model)
- **LLMs**:
  - [PolyCoder](https://huggingface.co/NinedayWang/PolyCoder-160M) for technical summaries
  - OpenAI GPT models for plain English explanations
